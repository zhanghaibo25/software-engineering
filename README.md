### 项目简介
热度校园是一款面向广大在校生，旨在激发在校大学生的学习氛围，学习激情，方便在校生参加各种社团活动。
用户可以通过该APP了解学校的空教室，了解学校的社团，学校的活动及其参加情况。通过APP上有关社团的介绍，学生
可以快速了解学校有哪些社团，可以了解社团的详细信息，找到志同道合的朋友。
